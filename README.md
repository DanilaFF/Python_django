# CRM бронирований (Django)

Учебный проект — CRM-система для управления клиентами и бронированиями, реализованная на Django.

##  Возможности
- управление клиентами
- создание бронирований
- просмотр списка броней
- отмена брони (логическое удаление через статус)
- поддержка формата даты `dd.mm.yyyy`
- авторизация пользователей
- административная панель Django

---

## Структура проекта
```

Python_django/
├── crm/                # основное приложение CRM
│   ├── models.py       # модели Client и Reservation
│   ├── views.py        # логика приложения
│   ├── forms.py        # формы Django
│   ├── urls.py         # маршруты CRM
│   ├── templates/      # HTML-шаблоны
│   └── migrations/     # миграции БД
│
├── myproject/          # конфигурация проекта Django
├── manage.py
├── .gitignore
└── README.md

````

---

## CRUD в проекте
- **Create** — создание клиента и бронирования
- **Read** — просмотр списка клиентов и броней
- **Update** — отмена бронирования (смена статуса)
- **Delete** — не используется

---

## Модели

### Client
- имя
- телефон
- заметки

### Reservation
- клиент
- дата
- время начала и окончания
- номер стола
- количество гостей
- статус (`booked`, `canceled`)

## Запуск проекта

### 1. Клонировать репозиторий
```bash
git clone <repo_url>
cd Python_django
````

### 2. Установить зависимости

```bash
pip install django
```

### 3. Применить миграции

```bash
python manage.py makemigrations
python manage.py migrate
```

### 4. Создать суперпользователя

```bash
python manage.py createsuperuser
```

### 5. Запустить сервер

```bash
python manage.py runserver
```
